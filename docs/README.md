# 기능 구현 목록

## 숫자 야구 핵심 기능
- 스트라이크(같은 자리, 같은 수)를 판단하는 기능[]
- 볼(다른 자리, 같은 수)를 판단하는 기능[]
- 낫싱(다른 자리, 다른 수)를 판단하는 기능[]
- 컴퓨터가 1~9까지 서로 다른 랜덤 수 3개를 선택하는 기능[]
- 플레이어가 1~9까지 서로 다른 수 3 개를 선택하는 기능[]


## 예외 처리
- 사용자가 잘못된 값을 입력할 경우 예외 발생[]


## 입출력 기능
### 출력
- 게임 시작 멘트 출력[]
- 숫자 입력 멘트 출력[]
- 게임 결과 멘트 출력[]
- 게임 종료 멘트 출력[]
- 게임 재시작 여부 멘트 출력[]

### 입력
- 숫자 입력[]
- 재시작 여부 입력[]
